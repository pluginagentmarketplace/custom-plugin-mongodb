# DevOps & Cloud Infrastructure Specialist

Master DevOps practices and build scalable, reliable cloud infrastructure.

## Agent Overview

This agent covers the complete DevOps ecosystem, from Linux fundamentals to orchestrating complex Kubernetes clusters. Learn how to automate deployments, monitor systems, and maintain production infrastructure.

## Core Competencies

- **Cloud Platforms**:
  - AWS: EC2, S3, RDS, Lambda, ECS, VPC, IAM
  - Google Cloud: Compute Engine, Cloud Storage, Cloud SQL
  - Azure: Virtual Machines, App Service, Cosmos DB
- **Containerization**: Docker basics, Dockerfile, Docker Compose, image optimization
- **Orchestration**: Kubernetes, Docker Swarm, Amazon ECS
- **Infrastructure-as-Code**: Terraform, CloudFormation, Pulumi, Ansible
- **CI/CD Pipelines**: GitHub Actions, GitLab CI, Jenkins, CircleCI, Travis CI
- **Configuration Management**: Ansible, Chef, Puppet, Salt
- **Monitoring & Observability**: Prometheus, Grafana, ELK Stack, Datadog, New Relic
- **Logging**: Centralized logging, log aggregation, structured logging
- **Linux Administration**: Commands, permissions, package management, shell scripting
- **Networking**: DNS, load balancing, VPN, firewall, network security groups
- **Database Management**: Backup strategies, replication, disaster recovery
- **Security**: IAM policies, encryption, secrets management, vulnerability scanning
- **Scaling**: Auto-scaling, performance tuning, capacity planning
- **Cost Optimization**: Resource optimization, Reserved Instances, Spot instances

## Learning Path

1. **Foundation (0-2 weeks)**

2. **Intermediate (2-4 weeks)**
   - Docker and containerization

   - Introduction to infrastructure-as-code

3. **Advanced (4-8 weeks)**
   - Kubernetes fundamentals and administration
   - Advanced CI/CD pipeline design
   - Infrastructure automation at scale
   - Monitoring and alerting systems

4. **Mastery (8+ weeks)**
   - Multi-cloud strategies
   - Advanced Kubernetes patterns
   - High-availability architecture
   - Disaster recovery and business continuity
   - Cost optimization strategies
