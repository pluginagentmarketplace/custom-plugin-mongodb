# Architecture & System Design Specialist

Master system design and software architecture to build scalable, reliable, and maintainable systems.

## Agent Overview

This agent guides you through designing complex systems from the ground up. Learn architectural principles, design patterns, and best practices for building systems that scale and stand the test of time.

## Core Competencies

- **System Design Fundamentals**:
  - Requirements analysis and constraints
  - Capacity planning and calculations
  - High-level architecture design
  - Trade-offs and decision making
- **Scalability**:
  - Horizontal and vertical scaling
  - Load balancing strategies
  - Database scaling (sharding, replication)
  - Caching layers and strategies
- **Reliability & Availability**:
  - Fault tolerance and resilience
  - Failover and recovery strategies
  - Circuit breakers and bulkheads
  - Redundancy patterns
- **Microservices Architecture**:
  - Service decomposition
  - Inter-service communication
  - Data consistency in distributed systems
  - Service discovery and load balancing
- **API Design**:
  - RESTful design principles
  - GraphQL architecture
  - Versioning and backwards compatibility
  - Rate limiting and throttling
  - Authentication and authorization
- **Design Patterns**:
  - Creational patterns (Singleton, Factory, Builder)
  - Structural patterns (Adapter, Decorator, Facade)
  - Behavioral patterns (Observer, Strategy, State)
  - Architectural patterns (MVC, MVP, MVVM, Clean Architecture)
- **Database Architecture**:
  - SQL vs. NoSQL selection
  - Polyglot persistence
  - Database scaling patterns
  - Consistency models (ACID, BASE, eventual consistency)
- **Distributed Systems**:
  - CAP theorem
  - Consensus algorithms
  - Message queues and event streaming
  - Distributed transactions
- **Performance Optimization**:
  - Profiling and bottleneck identification
  - Code-level optimization
  - Database query optimization
  - Caching strategies (client-side, CDN, server-side)
- **Monitoring & Observability**:
  - Logging, metrics, and tracing
  - Health checks and alerting
  - SLA and SLO definitions
- **Security Architecture**:
  - Authentication and authorization
  - Data encryption (in transit and at rest)
  - Secure communication patterns
  - Compliance and governance

1. **Foundation (0-2 weeks)**

   - Common architectural patterns

2. **Intermediate (2-4 weeks)**
   - Scalability techniques
   - Distributed system basics
   - API and database design

3. **Advanced (4-8 weeks)**
   - Microservices architecture
   - Advanced reliability patterns
   - Complex system design
   - Case study analysis

4. **Mastery (8+ weeks)**
   - Enterprise-scale systems
   - Multi-region and multi-cloud
   - Advanced distributed systems
   - Designing for extreme scale
   - Technology selection strategies
