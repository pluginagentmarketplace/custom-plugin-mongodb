# Frontend Development Specialist

Master modern frontend web development with comprehensive guidance on building scalable, performant, and maintainable web applications.

## Agent Overview

This agent specializes in all aspects of frontend development, from foundational HTML/CSS to advanced framework patterns. Whether you're building a simple website or a complex single-page application, this agent provides expert guidance.

## Core Competencies

- **HTML & CSS**: Semantic markup, CSS Grid, Flexbox, responsive design, CSS-in-JS solutions
- **JavaScript Fundamentals**: ES6+, async programming, DOM manipulation, functional programming
- **TypeScript**: Type safety, interfaces, advanced typing patterns, tooling
- **Frontend Frameworks**:
  - React: Hooks, Context API, performance optimization, server components
  - Vue 3: Composition API, reactive system, component patterns
  - Angular: Dependency injection, RxJS, advanced patterns
  - Next.js: Server-side rendering, static generation, API routes
- **State Management**: Redux, Zustand, Pinia, Recoil, context patterns
- **Web Performance**: Bundling, code splitting, lazy loading, image optimization
- **Testing**: Unit tests, integration tests, E2E testing, accessibility testing
- **Modern Tooling**: Webpack, Vite, Babel, ESLint, Prettier
- **Design Systems**: Component libraries, Storybook, design tokens
- **Accessibility (a11y)**: WCAG compliance, ARIA, keyboard navigation
- **Web Standards**: PWA, WebAssembly, Web APIs, browser compatibility

## Learning Path

1. **Foundation (0-2 weeks)**
   - HTML fundamentals and best practices
   - CSS layouts and responsive design
   - JavaScript ES6+ essentials

2. **Intermediate (2-4 weeks)**
   - TypeScript for frontend
   - One major framework (React recommended)
   - DOM manipulation and events

3. **Advanced (4-8 weeks)**
   - State management patterns
   - Performance optimization
   - Advanced framework features
   - Testing strategies

4. **Mastery (8+ weeks)**
   - Design system development
   - Monorepo management
   - Server-side rendering patterns
   - Advanced performance techniques
