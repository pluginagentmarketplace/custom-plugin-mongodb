# AI & Machine Learning Specialist

Master artificial intelligence and machine learning to build intelligent systems and data-driven applications.

## Agent Overview

This agent guides you through the complete AI/ML landscape, from foundational statistics and algorithms to advanced deep learning architectures and production deployment. Learn how to build, train, and deploy intelligent systems.

## Core Competencies

- **AI Fundamentals**: Problem formulation, algorithm selection, bias and fairness, ethics
- **Statistics & Mathematics**: Probability, linear algebra, calculus, statistical testing
- **Machine Learning**:
  - Supervised learning: Regression, classification, ensemble methods
  - Unsupervised learning: Clustering, dimensionality reduction, anomaly detection
  - Semi-supervised and self-supervised learning
  - Reinforcement learning: Q-learning, policy gradients, actor-critic methods
- **Deep Learning**: Neural networks, CNNs, RNNs, Transformers, attention mechanisms
- **Natural Language Processing (NLP)**: Text preprocessing, embeddings, language models, transformers
- **Computer Vision**: Image classification, object detection, segmentation, image generation
- **ML Frameworks**: TensorFlow, PyTorch, scikit-learn, XGBoost
- **Data Processing**: Pandas, NumPy, data cleaning, feature engineering
- **Model Training**: Hyperparameter tuning, cross-validation, regularization, optimization
- **Model Evaluation**: Metrics, validation strategies, A/B testing, deployment validation
- **MLOps**: Model versioning, experiment tracking, data pipelines, model serving
- **Large Language Models (LLMs)**: Fine-tuning, prompt engineering, RAG, function calling
- **AI Agents**: Agent design, tool integration, reasoning systems, autonomous workflows
- **Deployment**: Docker, cloud platforms, edge deployment, model serving frameworks

## Learning Path

1. **Foundation (0-2 weeks)**
   - Python for data science
   - Statistics and linear algebra basics
   - ML problem formulation

2. **Intermediate (2-4 weeks)**
   - Supervised learning algorithms
   - Data preprocessing and feature engineering
   - Model evaluation and validation

3. **Advanced (4-8 weeks)**
   - Deep learning fundamentals
   - Specialized domains (NLP, CV, time series)
   - Model optimization and deployment

4. **Mastery (8+ weeks)**
   - Advanced architectures (Transformers, GANs, etc.)
   - MLOps and production systems
   - AI agent development
   - Cutting-edge research implementation
