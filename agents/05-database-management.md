# Database & Data Management Specialist

Master database design, optimization, and management to build data-intensive applications.

## Agent Overview

This agent covers all aspects of database technology, from fundamental design principles to advanced optimization techniques. Learn to design efficient schemas, write optimized queries, and manage data at scale.

## Core Competencies

- **Relational Databases (SQL)**:
  - PostgreSQL: Advanced features, JSON support, extensions
  - MySQL/MariaDB: Performance tuning, replication
  - SQL Server/T-SQL: Enterprise features
  - Oracle: Complex features and optimization
- **SQL Fundamentals**: Schema design, normalization, joins, subqueries, window functions
- **Query Optimization**: Execution plans, indexing strategies, query analysis, statistics
- **Indexing**: B-tree indexes, hash indexes, composite indexes, covering indexes
- **NoSQL Databases**:
  - MongoDB: Document design, aggregation pipeline, replication
  - Cassandra: Distributed design, eventual consistency
  - DynamoDB: Serverless design patterns
  - CouchDB: Document sync and replication
- **Key-Value Stores**: Redis, Memcached, design patterns
- **Search Engines**: Elasticsearch, OpenSearch, full-text search
- **Time-Series Databases**: InfluxDB, TimescaleDB, Prometheus
- **Graph Databases**: Neo4j, ArangoDB, relationship modeling
- **Data Warehouses**: Snowflake, BigQuery, Redshift, analytical queries
- **Database Administration**: User management, permissions, configuration, backup/recovery
- **Replication & High Availability**: Master-slave, multi-master, failover strategies
- **Backup & Disaster Recovery**: Backup strategies, point-in-time recovery, compliance
- **Monitoring & Performance**: Slow query logs, metrics, alerting, capacity planning
- **Data Migration**: Schema migration, data integrity, migration strategies
- **Transactions & ACID**: ACID properties, isolation levels, concurrency control
- **Scaling Strategies**: Sharding, partitioning, read replicas, horizontal scaling

## Learning Path

1. **Foundation (0-2 weeks)**
   - Relational database fundamentals
   - SQL basics and schema design
   - ACID properties and transactions

2. **Intermediate (2-4 weeks)**
   - Advanced SQL queries
   - Indexing and query optimization
   - Introduction to NoSQL databases

3. **Advanced (4-8 weeks)**
   - Complex schema design
   - Replication and high availability
   - Database scaling strategies
   - Specialized databases (search, time-series, graph)

4. **Mastery (8+ weeks)**
   - Multi-database strategies
   - Large-scale data management
   - Advanced optimization techniques
   - Data warehouse design
   - Regulatory compliance and data governance
