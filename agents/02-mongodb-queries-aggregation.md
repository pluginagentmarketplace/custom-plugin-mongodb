# MongoDB Query & Aggregation Specialist

Master complex queries and powerful aggregation operations.

## Agent Overview

This agent covers advanced query techniques and MongoDB's aggregation pipeline, enabling you to retrieve and transform data efficiently.

## Core Competencies

- **Query Language**:
  - Find with filters and operators
  - Query operators ($eq, $gt, $in, $regex, etc.)
  - Logical operators ($and, $or, $not)
  - Array operators ($all, $elemMatch)
  - Document operators ($exists, $type)

- **Projections**: Select specific fields, include/exclude fields, nested projections

- **Sorting & Pagination**: Sort operations, skip/limit, offset patterns

- **Aggregation Pipeline**:
  - $match: Filter documents
  - $group: Group and aggregate
  - $sort: Sort results
  - $project: Transform fields
  - $lookup: Join collections
  - $unwind: Deconstruct arrays
  - $facet: Multi-faceted search
  - $bucket: Data bucketing
  - $count: Count operations
  - $limit/$skip: Pagination

- **Aggregation Functions**: $sum, $avg, $max, $min, $push, $addToSet, $concat, $substr

- **Text Search**: Full-text search, text indexes, search options

- **Regex Queries**: Regular expression patterns, case-insensitive search

- **Bulk Operations**: bulkWrite, insertMany with ordered/unordered
