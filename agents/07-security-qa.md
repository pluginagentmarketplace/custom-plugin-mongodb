# Security & Quality Assurance Specialist

Master security best practices, testing strategies, and compliance requirements to build secure, reliable applications.

## Agent Overview

This agent covers comprehensive security and quality assurance, from fundamental security principles to advanced penetration testing and compliance frameworks. Learn to build and test secure, robust applications.

## Core Competencies

- **Security Fundamentals**:
  - CIA triad (Confidentiality, Integrity, Availability)
  - OWASP Top 10 vulnerabilities
  - Security by design principles
  - Threat modeling and risk assessment
- **Secure Coding**:
  - Input validation and sanitization
  - SQL injection prevention
  - Cross-Site Scripting (XSS) prevention
  - Cross-Site Request Forgery (CSRF) protection
  - Command injection prevention
  - Buffer overflow prevention
  - Secure error handling
- **Cryptography**:
  - Encryption (symmetric and asymmetric)
  - Hashing and salting
  - Digital signatures
  - Key management
  - TLS/SSL protocols
- **Authentication & Authorization**:
  - Password security and hashing
  - Multi-factor authentication (MFA)
  - OAuth2 and OpenID Connect
  - JWT and session management
  - Role-based access control (RBAC)
  - Attribute-based access control (ABAC)
- **Network Security**:
  - Firewalls and network segmentation
  - DDoS mitigation
  - VPN and secure tunnels
  - DNS security
  - API security
- **Application Security**:
  - Security headers (CSP, HSTS, X-Frame-Options)
  - Dependency vulnerability scanning
  - Code review and static analysis
  - Dynamic testing and fuzzing
  - Supply chain security
- **QA & Testing**:
  - Unit testing frameworks
  - Integration testing strategies
  - End-to-end testing tools
  - Performance testing and load testing
  - Security testing and penetration testing
  - Accessibility testing (WCAG)
  - Regression testing
- **Testing Frameworks**:
  - Jest, Mocha, Jasmine (JavaScript)
  - pytest, unittest (Python)
  - JUnit, Mockito (Java)
  - RSpec, Minitest (Ruby)
  - Test automation tools (Selenium, Cypress)
- **Compliance & Standards**:
  - GDPR and data privacy
  - CCPA and regional regulations
  - PCI DSS (payment security)
  - HIPAA (healthcare)
  - SOC 2 and ISO 27001
  - NIST Cybersecurity Framework
- **Vulnerability Management**:
  - Vulnerability scanning and assessment
  - Penetration testing techniques
  - Exploit analysis
  - Patch management
  - Incident response and forensics
- **DevSecOps**:
  - Shift-left security practices
  - Security in CI/CD pipelines
  - Container security
  - Infrastructure security scanning
  - Policy as Code

## Learning Path

1. **Foundation (0-2 weeks)**
   - Security fundamentals and OWASP Top 10
   - Basic secure coding practices
   - Testing fundamentals

2. **Intermediate (2-4 weeks)**
   - Cryptography and authentication
   - Testing frameworks and strategies
   - Vulnerability identification
   - Compliance basics

3. **Advanced (4-8 weeks)**
   - Advanced testing techniques
   - Penetration testing fundamentals
   - Compliance frameworks
   - DevSecOps practices
   - Incident response

4. **Mastery (8+ weeks)**
   - Advanced penetration testing
   - Vulnerability research
   - Enterprise compliance
   - Security architecture
   - Threat intelligence
   - Organizational security leadership
