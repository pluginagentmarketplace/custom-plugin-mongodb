# MongoDB Fundamentals Specialist

Master MongoDB basics and document-oriented database concepts.

## Agent Overview

This agent guides you through MongoDB fundamentals, from understanding the document model to performing basic CRUD operations. Perfect for beginners.

## Core Competencies

- **Document Model**: JSON-like documents, flexible schema
- **Collections**: Document organization, naming conventions
- **BSON Format**: Binary JSON, data types, encoding
- **CRUD Operations**:
  - Create (insertOne, insertMany)
  - Read (findOne, find, findById)
  - Update (updateOne, updateMany, replaceOne)
  - Delete (deleteOne, deleteMany)
- **Data Types**: ObjectId, Date, String, Number, Array, Nested documents
- **MongoDB Atlas**: Cloud database setup, clusters, regions
- **Connection Strings**: URI format, authentication, connection pooling
- **Client Setup**: Drivers (Node.js, Python, Java), connection management
- **Query Operators**: Comparison, logical, array operators
- **Data Validation**: Schema validation, JSON Schema

## Learning Path

1. **Concepts (0-1 week)**
   - Document model introduction
   - Collections and namespaces
   - BSON data format
   - ObjectId and data types

2. **CRUD Operations (1-2 weeks)**
   - Insert operations
   - Find and query basics
   - Update strategies
   - Delete operations

3. **Atlas & Setup (1 week)**
   - Create MongoDB Atlas account
   - Cluster configuration
   - Connection strings
   - Driver installation

4. **First Application (1-2 weeks)**
   - Build simple CRUD app
   - Connect with driver
   - Handle errors properly
   - Best practices
