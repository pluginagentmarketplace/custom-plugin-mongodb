# /assess-skills - Assess Your Knowledge Level

Evaluate your current knowledge across different specializations and get personalized recommendations.

## Description

Take a quick assessment to understand your skill level and receive tailored learning recommendations.

## Usage

```
/assess-skills [specialization]
```

- Modern frameworks (React, Vue, Angular)

- Performance and accessibility
- Testing and component design

### Backend Development

- Async programming patterns
- Scalability and caching

### DevOps & Infrastructure
- Docker and containerization
- Kubernetes orchestration
- CI/CD pipelines

- Monitoring and logging

### AI & Machine Learning
- ML fundamentals and algorithms

- Model training and deployment
- LLMs and prompt engineering

### Database Management

- Schema design

### System Architecture

### Security & QA

## Assessment Levels

### Beginner
- 0-6 months experience
- Foundational knowledge
- Learning fundamentals
- Focus: Theory and basics

### Intermediate
- 6 months - 2 years experience
- Practical experience
- Building applications
- Focus: Best practices

### Advanced
- 2-5 years experience
- Complex systems
- Architectural decisions
- Focus: Optimization and scaling

### Expert
- 5+ years experience
- Leadership responsibilities
- Research and innovation
- Focus: Strategic decisions

## Assessment Process

1. **Quick Quiz**: 10-15 questions per specialization
2. **Skill Matrix**: See your strengths and areas for improvement
3. **Recommendations**: Personalized learning paths
4. **Resources**: Curated resources for gap areas

## Sample Assessment

```
/assess-skills frontend

Assessing your Frontend Development skills...

Q1: Which of these is NOT a JavaScript ES6+ feature?
Q2: Explain the difference between React hooks and class components
Q3: What is the purpose of the useEffect hook?
...
```
