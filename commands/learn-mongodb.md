# /learn-mongodb - Start Your MongoDB Learning Journey

Begin a structured MongoDB learning path tailored to your level and goals.

## Description

Choose your starting point and follow a guided learning path through MongoDB fundamentals to advanced topics.

## Usage

```
/learn-mongodb
```

## Learning Levels

### Beginner Path (0-2 weeks)
Master MongoDB basics and fundamental concepts.

**Topics:**
- Document model and BSON format
- MongoDB Atlas setup
- Basic CRUD operations
- Collections and queries
- First MongoDB application

**Outcome:** Can perform basic CRUD and write simple queries

### Intermediate Path (2-4 weeks)
Develop practical MongoDB development skills.

**Topics:**
- Complex queries and filtering
- Aggregation pipeline basics
- Schema design patterns
- Indexes and basic optimization
- Application integration

**Outcome:** Can build production-ready CRUD applications

### Advanced Path (4-8 weeks)
Master advanced MongoDB concepts.

**Topics:**
- Complex aggregation pipelines
- Schema patterns and normalization
- Performance optimization
- Transactions and sessions
- Replication and sharding basics

**Outcome:** Can optimize queries and design efficient schemas

### Expert Path (8+ weeks)
Specialize in MongoDB architecture and operations.

**Topics:**
- Replication and replica sets
- Sharding and distributed systems
- Security and encryption
- Backup and disaster recovery
- Production operations
- High availability design

**Outcome:** Can architect and manage production MongoDB systems
